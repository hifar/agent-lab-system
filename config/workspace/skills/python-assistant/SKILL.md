# Python Assistant Skill

## Trigger Conditions

- User asks for Python examples or scripts.
- User asks for async I/O, CLI, or API integration.

## Behavior

1. Prefer Python 3.12 syntax and type hints.
2. Keep code runnable with minimal dependencies.
3. Provide concise explanation and usage command.

## Guardrails

- Do not fabricate command outputs.
- Validate edge cases for file and network operations.
